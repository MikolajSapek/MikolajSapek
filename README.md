## Highlights

- At **REDD**, I built an automated news analysis system that reduced daily processing from **36 hours to 20 minutes**
- **Banana Ripeness Classification** — a custom CNN trained from scratch beats a frozen MobileNetV2 benchmark at **98.0% test accuracy**, with leak-aware (pHash) splitting and Grad-CAM ([repo](https://github.com/MikolajSapek/banana-ripeness-classification))
- **Echo Chamber Linguistics** — predicting political channel ideology from **301k YouTube comments**; five models from TF-IDF to fine-tuned DistilBERT, with a cross-channel generalisation study ([repo](https://github.com/MikolajSapek/echo-chamber-linguistics))
- **Forecasting Danish Pharmaceutical Exports** — a diagnostics-first 24-month SARIMA vs ETS forecast through the GLP-1 boom, validated with rolling cross-validation ([repo](https://github.com/MikolajSapek/danish-pharma-exports-forecast))
- Bachelor thesis: analyzed third-force candidates' impact on voter turnout in Polish presidential elections using county-level PKW data, Welch's t-tests, and linear regression
- Built **Apron** — a B2B shift management platform for Denmark (Next.js, Supabase, PostGIS)
